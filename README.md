# TireGuard AI: ระบบตรวจสอบสภาพยางรถยนต์อัจฉริยะ (Smart Tire Classification System) 🛡️🛞

โปรเจกต์วิทยานิพนธ์นี้เป็นการพัฒนาแบบจำลองปัญญาประดิษฐ์เพื่อจำแนกคุณภาพของยางรถยนต์ โดยใช้สถาปัตยกรรมโครงข่ายประสาทเทียมระดับลึก (CNN) เพื่อช่วยเพิ่มความปลอดภัยและมาตรฐานในการตรวจสอบหน้ายาง 

## 🌟 คุณสมบัติเด่น (Main Features)
- **การจำแนกแบบ 3 หมวดหมู่ (3-Class Classification):** สามารถคัดแยก "ยางสภาพปกติ (Good)", "ยางชำรุด (Defective)" และระบบกรองข้อมูล "ไม่ใช่ยางรถยนต์ (Not Tire)"
- **ความแม่นยำสูง (High Accuracy):** ผลลัพธ์จากการทดลองด้วยสถาปัตยกรรม **ResNet50V2** ให้ค่าความแม่นยำสูงถึง **96.42%**
- **ความรัดกุมของระบบ (Robustness):** มีระบบคัดกรองวัตถุที่ไม่เกี่ยวข้อง (Out-of-Distribution Detection) เพื่อป้องกันความผิดพลาดในการประมวลผลสภาวะจริง
- **การใช้งานที่ยืดหยุ่น:** รองรับการทำงานทั้งบน Web Application และการเชื่อมต่อผ่านกล้องสมาร์ทโฟน

## 📊 ผลการทดลองโดยสรุป (Research Findings)
- **สเกลข้อมูล:** ชุดข้อมูลภาพยางทั้งหมด 2,165 ภาพ และชุดข้อมูลเสริม (Not Tire) 304 ภาพ
- **ประสิทธิภาพ:** ค่าความถูกต้องเฉลี่ย 94.07% (สำหรับระบบ 3 คลาส)
- **ผลลัพธ์ Not Tire:** มีความแม่นยำในการคัดกรองสูงถึง 100% (Precision = 1.00)

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)
- **Backend:** Python, Flask
- **Machine Learning:** TensorFlow, Keras, ResNet50V2
- **Frontend:** HTML5, Modern CSS (Glassmorphism), JavaScript
- **Deployment:** GitHub LFS (Recommended for Model hosting)

## 🚀 ขั้นตอนการติดตั้ง (Installation)
1. ติดตั้งไลบรารีที่จำเป็น:
   ```bash
   pip install -r requirements.txt
   ```
2. รันแอปพลิเคชัน:
   ```bash
   python app.py
   ```
3. เข้าชมระบบผ่านเบราว์เซอร์ไปที่ `http://127.0.0.1:5000`
